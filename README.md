# 上海图书馆检索 (shlib)

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

一个快速、简洁的上海图书馆图书检索工具，支持 CLI 命令行和 Web 界面两种使用方式。

**GitHub**: https://github.com/ZedeX/shanghai-library-book-search

## 功能特性

- 🔍 **快速搜索** - 支持书名、作者、出版社、索书号等多维度搜索
- 📚 **馆藏查询** - 实时查看各分馆馆藏与借阅状态
- ⚡ **极速响应** - 部署在 Cloudflare 边缘网络，全球加速
- 📍 **偏好图书馆** - 设置常用图书馆，优先显示该馆馆藏
- 📖 **索书号查询** - 直接按索书号定位图书位置
- 🌐 **全分馆覆盖** - 覆盖上海图书馆及16个区级分馆

## 安装

```bash
# 克隆仓库
git clone https://github.com/ZedeX/shanghai-library-book-search.git
cd shanghai-library-book-search

# 安装依赖
npm install

# 全局安装 CLI（可选）
npm link
```

## CLI 使用

### 基本用法

```bash
# 搜索图书（显示所有场馆可借情况）
shlib 红楼梦

# 搜索图书（指定场馆）
shlib 红楼梦 东馆

# 按书名搜索
shlib 三体 -t title

# 按作者搜索
shlib 刘慈欣 -t author

# 按出版社搜索
shlib 人民文学出版社 -t publisher

# 按索书号搜索
shlib I247.5 -t callnumber

# 查看第2页结果
shlib 红楼梦 -p 2

# 每页显示10条结果
shlib 红楼梦 -l 10

# 查看图书详情
shlib -d 4264088b-1b2c-4a76-9544-1823d8354d3f
```

### 参数说明

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| `--type` | `-t` | 搜索类型：all/title/author/publisher/subject/callnumber | all |
| `--page` | `-p` | 页码 | 1 |
| `--limit` | `-l` | 每页结果数（最大50） | 20 |
| `--detail` | `-d` | 直接查看图书详情 | - |
| `--help` | `-h` | 显示帮助信息 | - |
| `--version` | `-v` | 显示版本号 | - |

## Web 部署

### 本地开发

```bash
npm run dev
```

访问 http://localhost:8789

### 部署到 Cloudflare Workers

本项目配置为 **Cloudflare Workers** 部署模式，后端 API 和静态资源都由 Workers 处理。

#### 方式一：通过命令行部署（推荐）

1. **安装 Wrangler CLI**
   ```bash
   npm install -g wrangler
   ```

2. **登录 Cloudflare**
   ```bash
   wrangler login
   ```
   这会打开浏览器，按提示完成授权。

3. **部署**
   ```bash
   npm run deploy
   # 或
   wrangler deploy
   ```

4. **查看部署状态**
   ```bash
   wrangler deployments list
   ```

部署成功后，会获得一个 `*.workers.dev` 域名。

#### 方式二：GitHub Actions 自动部署

1. **获取 Cloudflare API Token**
   - 登录 Cloudflare Dashboard
   - 进入 "My Profile" > "API Tokens"
   - 点击 "Create Token"
   - 选择 "Edit Cloudflare Workers" 模板
   - 复制生成的 Token

2. **配置 GitHub Secrets**
   - 进入你的 GitHub 仓库
   - Settings > Secrets and variables > Actions
   - 添加以下 secrets:
     - `CLOUDFLARE_API_TOKEN`: 你的 API Token
     - `CLOUDFLARE_ACCOUNT_ID`: 你的 Cloudflare Account ID（在 Dashboard 右侧边栏可找到）

3. **创建 GitHub Actions 工作流**
   
   创建 `.github/workflows/deploy.yml`:
   ```yaml
   name: Deploy to Cloudflare Workers

   on:
     push:
       branches: [main]

   jobs:
     deploy:
       runs-on: ubuntu-latest
       steps:
         - uses: actions/checkout@v4

         - uses: actions/setup-node@v4
           with:
             node-version: '20'

         - run: npm ci

         - name: Deploy to Cloudflare Workers
           uses: cloudflare/wrangler-action@v3
           with:
             apiToken: ${{ secrets.CLOUDFLARE_API_TOKEN }}
             accountId: ${{ secrets.CLOUDFLARE_ACCOUNT_ID }}
   ```

4. **推送代码触发部署**
   ```bash
   git add .
   git commit -m "Add GitHub Actions deployment"
   git push
   ```

### 部署后验证

部署成功后，访问你的 Worker URL（如 `https://shlib.your-subdomain.workers.dev`），应该能看到搜索页面。

测试 API：
```bash
# 搜索
curl "https://your-worker.workers.dev/api/search?q=红楼梦"

# 详情
curl "https://your-worker.workers.dev/api/detail/4264088b-1b2c-4a76-9544-1823d8354d3f"
```

## API 接口

### 搜索图书

```
GET /api/search?q={keyword}&type={type}&page={page}&limit={limit}&library={library}
```

参数：
- `q` - 搜索关键词（必填）
- `type` - 搜索类型：all/title/author/publisher/subject/callnumber（默认：all）
- `page` - 页码（默认：1）
- `limit` - 每页结果数（默认：20）
- `library` - 按图书馆筛选

### 获取图书详情

```
GET /api/detail/{recordId}
```

### 获取馆藏信息

```
GET /api/holdings/{recordId}
```

### 获取完整记录（详情+馆藏）

```
GET /api/record/{recordId}
```

# 产品需求文档

## MRD - 市场需求文档

### 1. 市场背景

#### 1.1 问题陈述

上海图书馆是上海市最大的公共图书馆系统，拥有丰富的馆藏资源。然而，官方检索系统存在以下问题：

- **访问速度慢**：官方系统响应时间长，用户体验不佳
- **移动端适配差**：官方系统在移动设备上显示效果不理想
- **缺乏个性化**：无法设置偏好图书馆，每次都需要手动筛选
- **无命令行支持**：对于开发者和技术用户，缺乏高效的 CLI 工具

#### 1.2 目标用户

| 用户群体 | 特征 | 核心需求 |
|---------|------|---------|
| 普通读者 | 偶尔借书，不熟悉图书馆系统 | 快速查找图书位置和可借状态 |
| 学生群体 | 频繁使用图书馆资源 | 高效检索、批量查询 |
| 开发者/技术用户 | 习惯命令行操作 | CLI 工具、API 接口 |
| 研究人员 | 需要精确检索特定主题 | 高级搜索、索书号查询 |

#### 1.3 市场机会

- 上海图书馆年接待读者超过 **500万人次**
- 移动端访问占比逐年上升，已超过 **60%**
- 开发者群体对 API 和 CLI 工具有强烈需求

### 2. 产品定位

**定位**：轻量级、高性能的上海图书馆检索工具

**核心价值**：
- ⚡ **速度**：部署在边缘网络，毫秒级响应
- 🎯 **精准**：多维度搜索，精确匹配需求
- 📱 **适配**：完美支持移动端和桌面端
- 🔧 **灵活**：Web + CLI 双模式，满足不同场景

### 3. 竞品分析

| 维度 | 本项目 | 官方系统 | 其他第三方 |
|-----|-------|---------|-----------|
| 响应速度 | ⚡ 毫秒级 | 🐢 秒级 | 🚀 较快 |

## PRD - 产品需求文档

### 1. 功能需求

#### 1.1 核心功能

| 功能模块 | 优先级 | 描述 |

| 馆藏查询 | P0 | 实时显示各分馆馆藏状态和可借数量 |
| 图书详情 | P0 | 展示图书完整信息（ISBN、出版社、简介等） |
| 分页浏览 | P0 | 支持分页查看搜索结果 |
| 图书馆筛选 | P1 | 按图书馆过滤搜索结果和馆藏信息 |
| 偏好设置 | P1 | 设置常用图书馆，优先显示该馆馆藏 |
| CLI 工具 | P1 | 命令行检索工具，支持所有搜索功能 |
| API 接口 | P1 | RESTful API，供第三方集成 |

#### 1.2 功能详细说明

##### 1.2.1 搜索功能

```
输入：搜索关键词 + 搜索类型
输出：图书列表（书名、作者、出版社、索书号、可借状态）

搜索类型：
- 全文搜索（默认）：搜索所有字段
- 书名搜索：仅匹配书名
- 作者搜索：仅匹配作者
- 出版社搜索：仅匹配出版社
- 索书号搜索：按索书号精确查找
```

##### 1.2.2 馆藏查询

```
输入：图书记录 ID
输出：馆藏列表（馆藏地、条码号、索书号、状态）

状态类型：
- 可借阅：图书在馆且可借
- 已借出：图书被借出
- 流转中：图书在馆际调拨中
- 不可借：图书丢失或不可外借
```

##### 1.2.3 偏好图书馆

```
功能：用户可设置常用图书馆
存储：localStorage（浏览器本地存储）
效果：
- 搜索结果优先显示该馆馆藏状态
- 详情页默认筛选该馆馆藏
- 可一键切换显示全部馆藏
```

## 开发指南

### 本地开发环境搭建

```bash
# 1. 克隆仓库
git clone https://github.com/ZedeX/shanghai-library-book-search.git
cd shanghai-library-book-search

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 访问 http://localhost:8789
```

### 代码规范

- **TypeScript**: 严格模式，所有类型必须明确定义
- **命名规范**: 
  - 变量/函数: camelCase
  - 类/接口: PascalCase
  - 常量: UPPER_SNAKE_CASE
- **注释**: 所有公共函数必须有 JSDoc 注释

## 常见问题

### Q: 为什么搜索结果数量显示 "XXX+"？

A: 上海图书馆系统不返回精确的总数，我们通过分页信息估算，因此显示为近似值加 "+"。

### Q: 如何设置偏好图书馆？

A: 在搜索结果页或图书详情页，选择图书馆下拉框后会自动保存到浏览器本地存储。

### Q: CLI 工具支持哪些搜索类型？

A: 支持 all（全文）、title（书名）、author（作者）、publisher（出版社）、subject（主题）、callnumber（索书号）。

### Q: 为什么有些图书没有封面？

A: 封面图片来自上海图书馆系统，部分图书可能没有封面数据。
